# Malawi Culture App

A mobile app showcasing the rich cultures, tribes, traditional foods, dances, and practices of Malawi. Built using **Kotlin** and **Jetpack Compose** for Android.

## Features

- Browse Malawi tribes and their cultural information
- View traditional foods, dances, and practices
- Slideshow of images for each culture type (planned)
- Interactive UI with hover/press animations
- Offline-ready using local drawable resources

## Project Structure

- `data/repository` - Hardcoded tribe data
- `ui/components` - Reusable composables like `CultureCard`
- `ui/theme` - App theme, colors, typography
- `MainActivity.kt` - App entry point
